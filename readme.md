# Wallet gate

Wallet gate uniformizes the JSON-RPC calls of the crypto nodes. Useful for wallets and exchanges.

## Features

- Address generation

- Block notification

- Transaction notification

- Withdrawal requests

## Supported currencies

- Bitcoin

- Ethereum (geth)

- more coming soon
